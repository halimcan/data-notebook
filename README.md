Yukarıdaki adımları takip ederek doğru klasöre konumlandıktan sonra şu komutu çalıştır:

```bash
jupyter notebook
```

Birkaç saniye sonra varsayılan tarayıcın yeni bir sekme açmalı ve bulunduğun klasördeki dosyaları listeleyen Jupyter arayüzünü göstermeli. Eğer otomatik açılmazsa, terminale bak; şöyle bir şey yazıyor olmalı:

```
The Jupyter Notebook is running at:
[I ..... NotebookApp] http://localhost:8888/?token=.....
```

Bu URL’yi kopyalayıp tarayıcında yeni bir sekmeye yapıştırarak açabilirsin.

Daha sonra bu challenge için Notebook’u açmak için `notebook_101` dosyasına tıklayabilirsin.
Çalışırken jupyter notebook komutunun çalıştığı terminal sekmesini kapatma veya `Ctrl`-`C` yapma; aksi hâlde Jupyter kapanır ve çalışmaya devam edemezsin!
